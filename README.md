# 🔍 Findora

**Lost it. Found it. Findora.**

Findora is a full-stack Lost & Found web application designed for college campuses. It helps students report, track, and recover lost items efficiently with a secure claim verification system and admin monitoring.

---

## 🚀 Features

- 🔐 User Authentication (Signup/Login)
- 📦 Post Lost / Found Items
- 🖼️ Multiple Image Upload Support
- 📍 Location Selection with Map
- 🔎 Search Functionality
- 📊 User Dashboard
- ✅ Claim Request & Verification System
- 🔒 Contact Privacy (Masked until approved)
- 🏷️ Item Status Tracking (Returned / Not Returned)
- 👤 Profile Management
- 🛠️ Admin Panel for Monitoring Items & Claims

---

## 🧠 How It Works

1. Users can post lost or found items with images and location.
2. Other users can search and request claims.
3. Owner verifies claim before revealing full contact details.
4. Once item is returned, owner marks it as returned.
5. Admin can monitor all activity.

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Frontend:** HTML, CSS, Bootstrap  
- **Other:** JavaScript, Leaflet (Maps)
